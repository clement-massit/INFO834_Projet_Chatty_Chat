# INFO834_Projet_Chatty_Chat  


## How to install  

1. clone the repository  
```
git clone https://github.com/clement-massit/INFO834_Projet_Redis
```  
2. install dependencies  
```
npm install 
```

## How to run the app  
1. create a data folder (for Mongod server)  

2. launch a mongod server  
```
mongod --dbpath ./data
```  
3. run the server  
```
node index.js
```
